# face-detection
Simple face detection with OpenCV.  
Face detection is performed using a Haar cascade from OpenCV.

## Usage
$**python3**  face_det.py  input-video-path  out-video-path 
 
Example:  
$**python3**  face_det.py  test.mp4  test_facedetected.mp4
